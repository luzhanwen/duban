# 1. 找到你要用的供应商。
# 2. 只把这一段的 apiKey 填上。
# 3. 导入 TXT，读伴会自动填好 provider、Base URL、模型名和价格。
#
# 如果你填了多个 API Key，可以把下一行改成当前要默认使用的供应商：
# provider = deepseek
#
# provider 可选：anthropic、openai、deepseek、kimi、openai-compatible。
# API Key 只会保存到本机浏览器 IndexedDB。
